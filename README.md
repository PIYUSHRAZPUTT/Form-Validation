# Form Validation using JavaScript

A simple and responsive client-side form validation project built using **HTML**, **CSS**, and **JavaScript**.  
This project validates user input fields for **username**, **email**, **phone number**, **password**, and **confirm password** with real-time error messages.


## Features

- Username must start with an **uppercase letter** and be at least **5 characters** long  
- Email must not exceed **50 characters**  
- Phone number must be exactly **10 digits**  
- Password must:
  - Be at least **8 characters** long  
  - Contain **one uppercase letter**  
  - Contain **one number**  
  - Contain **one underscore (`_`)**  
  - Contain **one special character**
- Confirm Password must match the entered password  
- Displays clear and user-friendly error messages
